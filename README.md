
GDC Haskell
============


FIrst Setup
---------

```sh

cabal install --only-dependencies

```

How to run test
-------------

```sh

cabal run

```

How to add test
------------

1. Import homework file in Spec.hs
2. Add test case like other tests.
3. Push to gdc-haskell/homework repository
4. Share the update with study!
